# 일식검담 패치 노트

## 밸런스 패치 0.4.1 — 육검 조율

6단계 CPU끼리 동일 시드와 좌우 교대 조건으로 치른 480경기를 기준으로, 강한 검객의 안전한 압박은 줄이고 약한 검객은 고유한 승리 수단을 강화했습니다.

### 전체 결과

| 검객 | 패치 전 | 패치 후 | 변화 |
|---|---:|---:|---:|
| 아야메 | 69.4% | 51.9% | -17.5%p |
| 이와오 | 55.6% | 47.5% | -8.1%p |
| 하야테 | 25.6% | 51.9% | +26.3%p |
| 카게로 | 37.5% | 46.2% | +8.7%p |
| 무진 | 63.8% | 44.4% | -19.4%p |
| 지코 | 47.5% | 56.9% | +9.4%p |

최고·최저 승률 차이는 43.8%p에서 12.5%p로 감소했습니다. 이 수치는 CPU 성향을 포함한 자동 대전 지표이며 실제 이용자 대전 결과와는 다를 수 있습니다.

### 아야메

- 체력: 925 → 900
- 제비베기: 가드당했을 때 중공격·기술로 이어지지 않도록 변경. 적중 시 연계는 유지합니다.
- 가로물결 피해: 110 → 105
- 낙월단칼 피해: 235 → 225
- 월광발도: 발동 7F → 9F, 회복 18F → 21F, 전진 거리 감소

빠른 확정타는 유지하되, 막힌 약공격과 빗나간 발도에서 상대가 반격할 틈을 만들었습니다.

### 이와오

- 직접 수치 변경 없음

긴 사거리와 큰 후딜이라는 현재 성격이 목표 범위에 있어 다른 검객 조정만 반영했습니다.

### 하야테

- 체력: 925 → 975 — 진입 후 연속기를 완주할 생존력 보강
- 잽찌르기: 회복 9F → 8F, 피해 55 → 40
- 엇갈려베기: 발동 8F → 7F, 회복 16F → 15F, 피해 95 → 72
- 쌍월가르기: 발동 14F 유지, 회복 27F → 25F, 피해 190 → 160
- 제비파고들기: 발동 6F → 5F, 회복 17F → 15F, 피해 40 → 30, 기력 소모 35 → 50
- 제비파고들기는 적중하면 약·중공격으로 연속할 수 있고, 연속하지 않거나 빗나가면 뒤로 빠집니다.
- CPU는 기력이 충분할 때만 연계 대시를 선택하고, 4단계에서도 사용 빈도를 절제합니다.

짧은 단도 사거리 때문에 접근 전에 지던 문제를 줄였습니다. 단발 피해 대신 빠른 연속기로 누적 피해를 만들며, 고효율 진입·이탈에는 기력 절반을 걸어야 합니다.

### 카게로

- 사슬휘돌리기 발동: 12F → 11F
- 추격일격 발동: 22F → 21F
- 사슬낚아채기 발동: 10F → 9F, 피해 55 → 60

느린 존잉 성격은 유지하면서 사슬의 유효 거리에서 먼저 공간을 잡기 쉽게 했습니다.

### 무진

- 체력: 1200 → 1125
- 물결긋기 피해: 95 → 90
- 갈대허리베기 피해: 170 → 155
- 해벽가르기 피해: 330 → 310

로스터 최고 수준의 체력과 단발 피해가 동시에 적용되던 부분을 낮췄습니다. 느리고 묵직한 정체성과 최고 단발 강공격은 유지됩니다.

### 지코

- 체력: 1020 → 1060
- 허리치기 피해: 106 → 112
- 중단 찌르기 피해: 118 → 125
- 머리치기 전방 사거리: 142px → 150px
- 중단 찌르기 전방 사거리: 146px → 162px

손목치기의 속도는 건드리지 않고, 검도식 중단 압박이 성공했을 때의 보상과 죽도 끝이 닿는 거리를 높였습니다.

### 검증 조건

- 난이도: CPU 4단계
- 대진: 6인 전원 조합 15개
- 각 대진: 좌우 16경기씩, 총 32경기
- 총 경기: 480경기
- 제한 시간: 경기당 60초, 1선승
- 결정론적 고정 시드 사용

# 일식검담 (ECLIPSE BLADES) — 전투 프로토타입

가상의 군도 국가 "해원국"을 무대로 한 1대1 2D 무기 대전 게임.
한 번의 확신이 라운드를 뒤집는, 칼날 거리 싸움이 핵심입니다.

`build/web`에는 iPhone Safari를 포함한 최신 브라우저에서 바로 실행할 수 있는 단일 스레드 웹 빌드가 들어 있습니다.

- 엔진: **Godot 4.7** / GDScript
- 아트: 수묵(먹·붓) 절차 드로잉 — 화지 바탕 + 먹 실루엣 + 인장 주홍 한 색
- 전투 로직: **60 TPS 고정, 전부 정수 연산** → 같은 입력 로그 = 같은 결과 (리플레이/롤백 대비)

## 실행

```bash
# 게임
godot --path .

# 테스트 (219건)
godot --headless --path . --script res://tests/run_tests.gd

# 소크 100경기
... --script res://tests/run_tests.gd -- soak=100
```

또는 Godot 에디터에서 프로젝트를 열고 F5.

## GitHub Pages 배포

`main` 브랜치의 `build/web`이 변경되면 `.github/workflows/deploy-pages.yml`이 정적 웹 빌드를 GitHub Pages에 자동 배포합니다. Godot에서 새 웹 빌드를 내보낸 뒤 해당 폴더를 커밋하고 푸시하면 됩니다.

## 모바일 웹

- iPhone/Android에서는 가로 화면을 기준으로 하며, 세로로 들면 회전 안내가 전투를 잠시 멈춥니다.
- 노치·홈 인디케이터를 피한 터치 조작을 제공하고 설정에서 표시 여부와 크기(85~120%)를 바꿀 수 있습니다.
- 첫 실행에는 엔진(WASM)과 에셋을 내려받으며 설치는 필요 없습니다. 브라우저 정책상 첫 터치 뒤부터 소리가 납니다.
- 웹 패키지에는 사용하지 않는 전투 스킨을 넣지 않고, 한글 글꼴은 실제 게임 문자를 추린 OFL Noto Sans KR을 포함합니다.

## 조작

| 동작 | P1 | P2 (로컬 2인) | 게임패드 |
|---|---|---|---|
| 이동/가드 | ← → (뒤로 = 가드) | J / L | 스틱·십자 |
| 점프 | ↑ | 8 | 스틱↑·십자↑ |
| 아래 (오의 커맨드) | ↓ | K | ↓ |
| 약베기 | A | U | X |
| 중베기 | S | I | Y |
| 강베기 | D | O | B |
| 기술 | **A+S 동시** | U+I 동시 | A |
| 오의(사맥 3) | **Q** | **P** | LB |
| 일시정지 | ESC | — | START |

- **점프**: ↑ (전진/후퇴/중립). 공중에서 공격 버튼 = **공중 공격**(점프당 1회). 점프 인 → 지상 콤보로 연결.
- **잡기**: 근접에서 → + 중베기
- **정밀 방어**: 피격 직전 3프레임 안에 뒤 입력 — 칩 피해 0, 사맥 +1
- **사맥(0~3칸)**: 칼결 정타·정밀 방어·휘두르기 처벌로 획득
  - 1칸: 후딜 중 **뒤+기술** = 자세 취소
  - 3칸: **오의 키(Q/P/LB)** = 즉시 오의. 기존 **↓↘→+강+기술**도 유지
- **무진 파도 연계**: **↓↘→+약/중/강** = 잔파/겹파/해일파. 사맥이 3칸일 때 강 연계는 1칸을 소비해 강화기 **홍해일파**로 바뀝니다.
- **칼결(정타)**: 무기 끝 20%만 맞히면 피해 ×1.25 + 사맥. 칼끝 주홍 표시가 그 구간.
- **경합**: 같은 틱에 무기끼리 교차하면 상쇄. 강도가 높은 쪽이 이김.
- **공중 저글**: 공중의 상대를 때리면 떠오름 → 짧은 저글 콤보 (천장·콤보 보정으로 제한)
- 체력 1000 · 60초 · 2선승(BO3)

## 음악

- 메뉴: 60BPM 고토·샤쿠하치 중심의 여백 있는 테마
- 전투: 132BPM 타이코·샤미센 오스티나토 중심의 긴박한 전용 루프
- 위기: 저체력 30% 이하, 남은 10초, 매치포인트에서 같은 박자의 겹북·고음 현 변주로 0.45초 교차 전환

## 훈련 모드 키

숫자 `1` 리셋 · `2` 판정 박스 · `3` 프레임 정보 · `4` 더미 모드(서기/가드/정밀가드/CPU) · `5` 리플레이 저장 · `6` 재생

F1~F6도 같은 기능을 유지하지만, MacBook에서는 `fn` 조합이 필요하므로 숫자열 사용을 권장합니다.

## 초상화

`art/portraits/arin.png`, `art/portraits/daeru.png` 를 넣으면 캐릭터 선택 화면에 자동 표시됩니다.

## 구조 (기획서 §6)

```
autoload/   설정(키 리맵·볼륨·토글)·오디오·게임 상태
combat/     시뮬 코어: sim_c(상수) · combat_world(60TPS 결정론 시뮬) · move_data · fighter_data · registry
data/       fighters/*.tres, moves/*.tres — 모든 수치는 여기서 수정 (코드 하드코딩 없음)
ai/         Utility CPU + 의미 행동별 다중 프레임 입력 큐 (난이도 1~4)
ui/         화면·수묵 뷰 레이어 (시뮬과 완전 분리)
fx/         절차 생성 사운드 + 생성 스크립트
tests/      수용 기준(AC) 헤드리스 테스트
```
